** Git 설치하기
-https://git-scm.com/downloads 
설치 후 Git Bash 실행 후 Git Bash 창에서git 실행으로 설치를 확인한다.
컴퓨터에 설치한 Git과 연결할 로컬 저장소를 만들기 위해, 연결할 폴더에서 마우스 오른 쪽 버튼으로 [ Git Bash Here] 선택하면 Git Bash 창이 열리고 작업 폴더가 나타남.
Git Bash 창에서 $ git init 실행 --> 숨김 폴더인 [ . git] 폴더가 생성되고, 로컬 저장소가 된다. 
작업 폴더 옆에 (master) 표시가 나타남.
Commit 생성  -  it 에서는 생성된  각 버전을 Commit 이라고 한다.
